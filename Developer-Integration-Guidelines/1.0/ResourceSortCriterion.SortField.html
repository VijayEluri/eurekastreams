ResourceSortCriterion.SortField

ResourceSortCriterion ResourceSortCriterion.SortDirection

Description

The field to sort by.

Enumeration Values

NAME - Sort by the resource’s name

DATE_ADDED - Sort by the date the resource was added to the system

FOLLOWERS_COUNT - Sort by the number of people following the resource

UPDATES_COUNT - Sort by the number of updates the resource has
