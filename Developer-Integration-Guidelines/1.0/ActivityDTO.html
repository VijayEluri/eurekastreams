ActivityDTO

ActivityVerb Data Types

Description

Contains the componets of an activty.

Contents

Type Value Description
Number id Uniquely identifies the activity
ActivityVerb verb Describes the action on the activity (post,share)
BaseObjectType baseObjectType An enumeration of object types for activities
HashMap<String, String> baseObjectProperties Describes the properties for objects for an activty
StreamEntityDTO actor Person that put the activity in the stream
StreamEntityDTO originalActor For shared activity, the original author of the activity
StreamEntityDTO proxyActor
StreamEntityDTO destinationStream The stream in which the activity resides
String location
String mood
String annotation
Boolean starred True/False flag defining if the activity has been saved or not
Date postedTime Date/time stamp marking when the activity was posted
CommentDTO firstComment Data set representing the first comment for the activity
CommentDTO lastComment Data set representing the last comment for the activity
Boolean commentable True/false flag defining whether commenting is enabled for the activity
Number commentCount Number of comments posted to the activity
List<CommentDTO> comments List of commets for the activity
Boolean deletable True/false flag defining whether the current user may delete the activity
Boolean isDestinationStreamPublic True/false flag defining whether the stream where the activity resides is public or not
Boolean shareable Defines whether the activity is shareable or not
EntityType appType The application type for the activity
String appName The name of the application where the activity resides
Number appId The unique identifier for the application where the activity resides
String appSource The name of the source stream for the application where the activity resides
Boolean liked True/false flag defining whether the activityhas been liked or not
Number likeCount Number of times the activity was liked
List<PersonModelView> likers List of users who liked the activity
