PersonModelView

PostActivityRequest PagedSet

Datatype

Description

Date set that contains the parameters necessary to define a person

Contents

Type Value Description
Number entityId Unique identifier for the person
Set<PersonModelView.Role> roles List that defines the roles the user holds
Date dateAdded Date/time stamp for when the person was added to the system
String accountId The person’s operating system id
String openSocialId The person’s unique identifier within the OpenSocial social media data interchange
String title The person’s job title
Number parentOrganizationId The unique identifier for the person’s parent organization
String parentOrganizationShortName The short name for the person’s parent organization
String parentOrganizationName The full name of the person’s parent organization
String description The description for the person
Number followersCount The number of followers the person has
Number followingCount The number of people the person is following
Number groupsCount The number of groups the person is a member of
Number compositeStreamHiddenLineIndex
String displayName The person’s display name
String avatarId The unique identifier for the person’s avatar
Number avatarCropSize the size of the person’s avatar in square pixels
Number avatarCropX The width of the person’s avatar in pixels
Number avatarCropY The height of the person’s avatar in pixels
Number updatesCount The number of updates to the person’s stream since it was last refreshed
Number streamId The unique identifier for the person’s stream
HashSet<Number> optOutVideos A list identifying which of the getting started videos the person has opted out of
Boolean tosAcceptance True/False flag defining if the person has accepted the terms of service
Boolean commentable True/False flag defining if commenting is enabled on the person’s stream
Boolean streamPostable True/False flag defining id posting is enabled on the person’s stream
AuthenticationType authenticationType Defines the type of authentication being used for the person
String email The person’s email address
List<Number> relatedOrganizationIds List of identifiers for the organizations the person is a member of
Date lastAcceptedTermsOfService Date/time stamp for when the terms of service was accepted
Hashmap<String, String> additionalProperties List of additional properties for the person
Boolean accountLocked True/False flag defining if the person’s Eureka account has been locked
String lastName The person’s last name
String preferredName The person’s preferred name
String jobDescription The person’s job description
String workPhone The person’s work phone number
String cellPhone The person’s cell phone number
String fax The person’s fax number
List<OrganizationModalView> relatedOrganizations List of the organization data sets for the organizations the person is a member of
List<String> interests List of the person’s interest
String biography The person’s biography
String bannerId The unique identifer for the banner assocuated with the person’s profile
Number bannerEntityId
