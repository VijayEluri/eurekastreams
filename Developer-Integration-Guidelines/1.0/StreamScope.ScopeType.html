StreamScope.ScopeType

SystemSettings StreamEntityDTO

Description

An enumeration describing the scope of a stream view.

Enumeration Values

ALL - Scoped for all

PERSON - Scoped to a specific person

ORGANIZATION - Scoped to a specific organization; messages posted to people and groups in an organization’s hierarchy

GROUP - Scoped to a specific group

PERSONS_PARENT_ORGANIZATION - Scoped to a person’s parent organization

PERSONS_FOLLOWED_STREAMS - Scoped to all streams followed by a person; key refers to the person’s unique key

STARRED - Scoped to starred activities for current user

RESOURCE - Represents a resource
