CommentDTO

DomainGroupModelView BreadcrumbDTO

Datatype

Description

Contains the parameters to define a comment

Contents

Type Value Description
Number id Unique identifier for the comment
String authorAvatarId Unique identifier for the avatar of the comment’s author
Number authorId Unique identifier for the comment’s author
String authorAccountId The unique system identifier for the author
String authorDisplayName The display name of the comment’s author
String body The text of the comment
Date timeSent Date/Time stamp when the comment was posted
Number activityID Unique identifier for the activity the comment is for
Boolean deletable Determines if the comment can be deleted by the current user
