Follow.FollowerStatus

GetCurrentUserFollowingStatusRequest EntityType

Description

Enumeration for follow/following relationship status.

Enumerated Values

NOTSPECIFIED - Follower/Following relationship is not specified

DISABLED - Follower/Following relationship is not allowed

FOLLOWING - Follow/Following relationship is established

NOTFOLLOWING - Follower/Following relationship is not established
