LinkInformation

LongidToLookup Job

Datatype

Java Package Name

org.eurekastreams.server.domain.stream

Contents

Type Value Description
Number id Unique identifier for the link
String title The title associated with the link
String Source The source site for the link (URL)
String url The URL for the link
String largestImageUrl URL for the largets image associated with the link
Set<String> imageUrls List of URLs for images associated with the link
String description Description of the link
String selectedThumbnail The URL selected for the image for the link
Date created Date/time stamp when the link was posted
