OrganizationModelView

Page MembershipCriteria

Datatype

Java Package Name

org.eurekastreams.server.search.modelview

Contents

Type Value Description
Number entityId Unique identifier for the organization
Number parentOrganizationId Unique identifier for the organization’s parent organization
String name The organization’s name
String description The description for the organization
String url The URL to the organization’s website
Number followerCount The number of people following the organization
Number descendantEmployeeCount The number of employees in the system that are members of the organization (aggregate)
Number descendantGroupCount Number of groups that fall under the organization (aggregate)
Number updatesCount The number of updates to the organization’s steam since the last refresh
Number childOrganizationCount The number of other organizations that fall under the organization
String shortName The short name for the organization
String avatarId The unique identifier for the avatar for the organization
Number avatarCropSize The size of the avatar for the organization in square pixels
Number avatarCropX The width of the organizations avatar in pixels
Number avatarCropY The height of the organization’s avatar in pixels
String overview The organization’s overview
Number streamId The unique identifier for the organization’s stream
String bannerId The unique identifier for the organization’s banner image in the organization’s profile
Number bannerEntityId
Boolean allUsersCanCreateGroups True/False flag defining whether all users can create groups within the organization or not
List<PersonModelView> leaders List of data sets representing the leaders in the organization
List<PersonModelView> coordinators List of data sets representing the organization coordinators
